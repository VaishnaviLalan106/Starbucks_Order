# Coffee Shop Order Demo ☕

This is a small practice project I made for fun while learning programming.

I tried implementing a simple **coffee ordering system** in two ways:

- A **Python console version** using classes and object-oriented programming
- A **static web version** using HTML, CSS, and JavaScript

The goal was just to experiment with logic and basic UI.

---

## Project Versions

### 1. Python Version (Console)

A simple terminal-based coffee ordering system.

Features:
- Display coffee menu
- Add coffee to order
- View current order
- Calculate total price
- Checkout confirmation

Concepts used:
- Python classes
- Lists
- Loops
- User input
- Basic object-oriented programming

---

### 2. Web Version (Static Page)

A simple coffee shop interface built with HTML, CSS, and JavaScript.

Features:
- Coffee menu cards
- Add items to cart
- View order cart
- Calculate total price
- Checkout button
- Clear cart option

Technologies used:
- HTML
- CSS
- JavaScript

---


## Purpose

This project was created purely for **learning and experimenting with programming concepts**, especially:

- Python object-oriented programming
- JavaScript DOM manipulation
- Basic UI layout with HTML and CSS
